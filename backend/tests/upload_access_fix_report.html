🚨 Problem Identified

Issue: 500 Internal Server Error

URL: http://localhost:8080/nwrcontractregistry/backend/uploads/18.20251006-091516.pdf

Cause: Incompatible .htaccess directives causing Apache configuration conflicts

🔍 Root Cause Analysis:

  • The .htaccess file contained some directives that may not be compatible with all Apache configurations
  • Complex regex pattern in FilesMatch directive was causing parsing issues
  • Some security directives were conflicting with server settings
✅ Solution Applied

Fix: Updated .htaccess Configuration

Simplified and made the .htaccess configuration more compatible while maintaining security

❌ Before (Problematic):

Options -ExecCGI -Indexes AllowOverride None php_flag engine off <FilesMatch "^.*\.(?!pdf|doc|docx|txt|xlsx|xls).*$"> Order allow,deny Deny from all </FilesMatch>

✅ After (Fixed):

Options -Indexes <Files ~ "\.php$"> Order allow,deny Deny from all </Files> RemoveHandler .php .phtml .php3 .php4 .php5 <FilesMatch "\.(pdf|doc|docx|txt|xlsx|xls)$"> Order allow,deny Allow from all </FilesMatch>
